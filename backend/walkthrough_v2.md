# 워크쓰루 - 뉴스 노출 문제 및 서버 중단 해결

안녕하세요! 요청하신 뉴스 노출 문제와 서버 중단 현상을 확인하여 조치했습니다.

## ✅ 해결된 문제
1. **뉴스 미노출**: 한국어 번역 실패 시에도 영어 뉴스가 대신 보이도록 백엔드를 보완했습니다.
2. **서버 중단**: 프론트엔드 서버가 꺼져 있던 것을 다시 실행했습니다.

## 🛠 수정 내역
- **Backend (`analyzer.py`)**: `fetch_crypto_news` 함수에서 AI 번역 오류 발생 시 빈 화면이 나오는 대신 원본 뉴스를 출력하도록 수정했습니다.
- **Server**: 프론트엔드(Vite)와 백엔드(Uvicorn) 서버를 모두 가동 중입니다.

## 🚀 확인 방법
1. 브라우저에서 `http://localhost:5173`에 접속합니다.
2. 상단에 `LATEST NEWS` 섹션이 정상적으로 표시되는지 확인합니다.
3. 한국어 모드에서도 뉴스가 하나 이상 표시되는지 확인합니다.

추가로 궁금하신 점이나 다른 요청사항이 있으시면 말씀해주세요!
