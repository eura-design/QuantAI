# AI 매매 성과 실시간 상태 표시 기능 작업 순서

이 가이드는 AI 매매 성과의 현재 상태를 시각화하는 과정을 안내합니다.

## 1단계: 백엔드 상태 조회 로직 추가
`TradeRepository`에 현재 매매 상태를 판단하는 로직을 추가합니다. `OPEN` 상태가 가장 우선순위가 높고, 그 다음 `PENDING`, 아무것도 없으면 `IDLE`로 정의합니다.

## 2단계: API 응답 확장
`/api/trades/stats` API가 현재 상태 정보를 포함하여 클라이언트에 전달하도록 수정합니다.

## 3단계: 다국어 번역 추가
각 상태값에 매칭되는 텍스트를 `translations.js`에 추가하여 사용자 언어에 맞게 표시합니다.

## 4단계: UI 컴포넌트 반영 및 스타일링
`TradePerformance.jsx`에서 API의 상태 데이터를 받아와 타이틀 옆에 애니메이션이 적용된 배지로 표시합니다.
- **시장 관망중 (IDLE)**: 차분한 회색
- **지정가 대기중 (PENDING)**: 주의를 끄는 노란색
- **포지션 보유중 (OPEN)**: 활동 중임을 알리는 녹색 펄스
